# python_awsSDK

DynamoDB migration practice with python SDK.

export dynamoDB to json file, and import json file to dynamoDB.

Can add custom converting logic to migrate.

# ENV

-   `python 3.9.1`
-   `pip 21.0.1`

# How to Run

1. clone this repo
2. `$ pip install -r requirements.txt`
3. python app.py
